# Python - Object-relational mapping

In this project, you will link two amazing worlds: Databases and Python!
In the first part, you will use the module MySQLdb to connect to a MySQL 
database and execute your SQL queries.
In the second part, you will use the module SQLAlchemy 
(don’t ask me how to pronounce it…) an Object Relational Mapper (ORM).


##Concepts you need to know to complete these tasks

- How to connect to a MySQL database from a Python script
- How to SELECT rows in a MySQL table from a Python script
- How to INSERT rows in a MySQL table from a Python script
- What ORM means
- How to map a Python Class to a MySQL table
